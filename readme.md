# Marcin's dotfiles.

This repo contains my dotfiles and it's based on [Addy Osmani](https://github.com/addyosmani/dotfiles/)'s one.

## overview of files

#### Shell environement
- `.bash_aliases`
- `.bash_exports`
- `.bash_functions`
- `.bash_profile`
- `.bash_prompt`
- `.bashrc`

#### Vim Config
- `.vimrc`
- `.vim`


#### Git
- `.gitconfig`
- `.gitignore`


## Installation && update

```
git clone https://github.com/marcinm2h/dotfiles.git && cd dotfiles && bash ./sync.sh
```
